# 塔防特殊属性

| 属性符号            | 属性含义                               |
| ------------------- | -------------------------------------- |
| fireCount           | 火球目标数量（同时发出多少个火球）     |
| lightningChainCount | 闪电链数量（同时发出多少条闪电链）     |
| lightningBounce     | 闪电链的跳转次数                       |
| iceCount            | 冰霜新星数量（同时发出多少个冰霜新星） |
| iceRange            | 每个冰霜新星会造成多大范围的伤害       |
